Game Development Section

The Game Development Section at the UCC Coding and Algorithms Academy meets once every two weeks to discuss game concepts, aiming to develop a toolkit that club members can draw from in their own future endeavors in game creation.

William Trojniak

Game Development Section Head

Welcome to the Game Development section. I am William Trojniak, a Grade 10 student in Bremner's House.

During each section discussion, I will be demonstrating a certain aspect of a game that I'm developing and how I'm achieving such functionality. Members will then be encouraged to implement their own take on such an element into their own game. In the next discussion, members will then have the opportunity to show off how they implemented that functionality into their own game, highlighting how their take differed from the implementation demonstrated.

Setting Up Our Playspace - HTML Canvas

Oct 3, 2019

The first game that we create will be Snake, coded in JavaScript and utilising an HTML canvas.

Through of Snake's development, the basics of JavaScript will be learned, allowing for more ambitious projects in the future.

During this discussion, we will be creating our canvas and learning how JavaScript can be used to draw graphics on the canvas. This functionality will be used to draw a checkered background and our snake.

<html>

<canvas id="myCanvas" width="480" height="480" style="border:1px solid #000000;"></canvas>

</html>

<script>

const canvas = document.getElementById("myCanvas");

const ctx = canvas.getContext("2d");

ctx.fillStyle = "#00FF00";

for(var c = 0; c < 15; c++){

for(var r = 0; r < 15; r++){

ctx.fillRect(c*32, r*32, 32, 32);

}

}

</script>
