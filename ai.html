What is Machine Learning?

Sept. 26, 2019

The idea of Artificial Intelligence and Machine Learning has been toyed with for a long time. Many films on the big screen have been seen using AI and ML, like Terminator, Avengers: Age of Ultron, or all Iron Man films (Jarvis). They rarely go deep into how they came to be, but rather that these computers are suddenly able to think for themselves.

In 1959, Arthur Samuel gave this description of Machine Learning: The field of study that gives computers the ability to learn without being explicitly programmed.

This is in contrast to AI, where the computer is explicitly programmed to behave in a certain way. In Machine Learning, we simply give the computer an algorithm to learn for itself, through trial and error (like real humans). There are many ways that scientists are trying to make robots that can think like humans, however, most often an algorithm that is programmed like a human brain, is able to learn like a human. These are called Neural Networks.

Let's take a step back, however, becuase many algorithms, used frequently with professional algorithms, are much simpler and more logical and statistical.

Tom Mitchell, in 1998, defined a Machine Learning Problem: A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.

Heres an example. Have you ever noticed that you get spam much less frequently than before? A spam filter is an ML algorithm that filters out spam emails, based on what emails that are marked spam frequently look like. In this scenario, the task T is marking spam emails, the performance measure P is whether or not emails that aren't marked by the algorithm are then marked by the user (probability of email being spam), and experience E is the input of seeing a bunch of emails marked spam by the user, and then developing an algorithm to allow the computer to mark those emails spam by itself.

Just to clarify, the computer makes these algorithms itself, we simply tell it how to learn. You can think of your job, as a Machine Learning Engineer, to teach the computer how to learn, and it will learn by itself.
