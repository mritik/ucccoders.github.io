Projects & Activities


DiaMonde Diamond Tracker
by Jefferson Ding & Samantha Shulman (BSS), Mar 2020

DiaMonde, based on the Ethereum blockchain, solves a major problem in the diamond industry by making each diamond's history transparent.

InterGen - Service for Seniors
by Robbie Evans, Mar 2020

InterGen is a dynamic web platform that connects seniors in hospitals and senior homes, alone and lacking interaction, with high school volunteers.

Password Generator & Manager
by Jefferson Ding, Feb 2020

This Python/TKinter tool generates secure, 1024-bit RSA encrypted passwords and stores them securely on the local machine.

TileVania
by William Trojniak, Jan 2020

A fun adventure game, built using Unity, where the protagonist skillfully navigates their way through countless obstacles, traps, enemies and hazards.

by Leo Zhang, Jan 2020

This is an online workflow manager allowing you to create and maintain task lists. Tasks are stored by Google Account using Firebase to authenticate.

by Kevin Liu, Dec 2019

Using C++, Kevin coded a simplistic highway traffic simulator. A highway is divided into blocks and traffic flow between blocks is calculated.

The Four Sections

Our club covers four aspects of coding: Algorithms, AI/Machine Learning, Web Development and Game Development.

Members can be involved in any or even all of the four sections (i.e. participate in respective discussions and projects).

Algorithms

Run by Kevin Liu

During each section discussion, Kevin will first give a short introduction on a well-defined algorithmic topic. Then he will leave a typical problem for the topic for members to work over the next couple of weeks. At the next meeting, Kevin will first allow members to share their solutions for the problem before moving onto another topic. After a topic is done, a summary note and a solution to the previous weekly problem will be posted. In the weeks leading up to the CCC, the main focus will be weekly CCC problems to help members prepare.

AI and Machine Learning

Run by Nima Bidari

Each section discussion meeting is split into two parts: a theoretical exploration and a practical exploration of machine learning. In the first part, Nima will lead a discussion about both the current AI events/trends and a theoretical topic on machine learning. The second half will focus on a specific problem in the AI area, which the Club members will collaborate to solve over the next couple of weeks. The language to be used is either Python (Scikit Learn & Tensorflow) or Javascript (Tensorflow.js).

Web Development

Run by Ibrahim Fadel

During each section discussion, Ibrahim will look at some real-world problems associated with key concepts of web development. Members will be given the opportunity to try to solve these problems, then talk about the solutions and discuss the benefits of each. A heavy focus will be put on writing clean reusable code that anyone could easily understand. In some meetings Ibrahim will look at front-end frameworks such as React and certain libraries such as Bootstrap and explain how to use them.

Game Development

Run by William Trojniak

Every two months or so, William will be presenting a general theme around which the club members will design a game individually, focusing on a certain element of the game each meeting. These elements could include features such as character movement, health, or a score indicator. By the end of the two months, the goal for the club members is that by incorporating these elements into their own games, they have created something fully playable with their own creative takes on the original theme.
